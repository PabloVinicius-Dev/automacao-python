# 🖥️ RPA para Cadastro Automatizado em Sistema Integrado a Banco de Dados

Este projeto foi desenvolvido como prática aplicada durante minha graduação em Ciência da Computação, com foco em automação de processos, manipulação de dados e simulação de interação humano-computador.

A aplicação implementa um fluxo de RPA (Robotic Process Automation) utilizando Python para automatizar o cadastro de registros em um sistema conectado a banco de dados.

---

## 📚 Contexto Acadêmico

O projeto reforça conceitos fundamentais da computação, como:

- Manipulação e estruturação de dados  
- Automação de processos  
- Interação com interfaces gráficas  
- Controle de fluxo e sincronização de execução  
- Redução de complexidade operacional por meio de software  

---

## ⚙️ Tecnologias Utilizadas

- **Python 3.x**  
- **Pandas** — Processamento e tratamento de dados estruturados  
- **PyAutoGUI** — Automação de interface gráfica (GUI Automation)  
- **time** — Controle de execução e sincronização  

---

## 🔄 Fluxo de Execução

1. Leitura dos dados a partir de fonte estruturada (CSV ou Excel)  
2. Tratamento e organização das informações utilizando Pandas  
3. Execução automatizada do preenchimento de campos via PyAutoGUI  
4. Inserção dos registros no sistema com validação sequencial  

---

## 🎯 Objetivo

Aplicar conceitos computacionais para automatizar tarefas repetitivas, reduzindo erros humanos, aumentando eficiência e demonstrando a integração entre manipulação de dados e automação de interface.
